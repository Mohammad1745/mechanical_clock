# Mechanical Clock

#### A simple HH:mm:ss mechanical flip clock

[Live Site](https://mechanicalclock.netlify.app/)

#### Setup
```bash
# install dependencies
$ npm install

# build app
$ npm run build

#open file /dist/index.html
```

#### Applied
`Javascript` `Webpack` `SCSS` `Node`
